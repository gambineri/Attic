* cfc (
str2check,
forbidden_chars
)


This command checks the existence in str2check of at most one character contained in the forbidden_chars parameter and returns a boolean value.


Parameters

str2check
The string in which the search will be made.

forbidden_chars
The string containing the forbidden chars to look for.



Return Values

If any of the forbidden chars has been found, the command returns the string 'true', otherwise it returns the string 'false'.

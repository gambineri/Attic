* cycle (
tqrname
)


Defines the starting point of a cycle on the records in the tqr specified by name in the parameter. The *endcycle command is necessary, otherwise an unpredictable behaviour is expected.


Parameters

tqrname
The name of the (existing) tqr on which the cycle will be done.



Return Values

This command doesn't return any data.

Remarks

The cycle begins from the current cursor position; the user must set the desired position of the internal cursor by using the *setrec command.
Note that if the cursor is at the beginning of the tqr, making two identical and consecutive cycles without using the *setrec command, implies that the first cycle is executed on all rows, the second one is not executed, because the internal cursor is at the end of the rowset.

Example

*exeq(articles, "SELECT * FROM articles", 5, 10)
*cycle(articles)
*recval(articles, id)

*endcycle(articles)

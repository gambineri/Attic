*if( leftVal Operator rightVal )

*elsif( leftVal Operator rightVal )

*else()

*endif()




Conditional commands to avoid or permit the processing of blocks of code during the processing itself.


Parameters

leftVal
Some text which is the string representation of the left value of the operand.

Operator
The binary operator used to determine a boolean value. Supported operators:


== comparison operator
!= inequality operator
< less than operator
<= less than or equal operator
>= greater than operator
> greater than or equal to operator

rightVal
Some text which is the string representation of the right value of the operand.



Remarks

The use of elsif after else is actually permitted, although the result is absolutely unpredictable. In any case, we believe in your smart mind which, surely, has just understood the nonsense hidden in a similar construct.



Example

*if(1==1)
  Block 1<br>
  *if(1>=)
    This block will never be processed<br>
  *elsif(aa!=cc)
    Block 2<br>

    *if(abc >= ab)
      Block 3<br>
    *elsif(abc <= ab)
      This block will never be processed<br>
    *endif()

  *elsif(bb==bb)
    This block will never be processed<br>
  *else()
    This block will never be processed<br>
  *endif()
*endif()

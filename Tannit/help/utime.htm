* utime (
year,
month,
day,
hour,
min,
sec
)


The command transforms a string of the type yyyy-mm-dd hh:mm:ss in a string representing a universal time (seconds past midnight of 1.1.1970) .



Parameters

year
expressed with for digits.

month
expressed with 2 digits.

day
expressed with 2 digits.

hour
expressed with 2 digits.

min
expressed with 2 digits.

sec
expressed with 2 digits.



Return Values

The corresponding universal time which represents the date-time passed to the command.


Example

*dmytime(*utime(2000, 7, 12, 12, 00, 50))

This produces the output 2000-07-12 12:00:50.

* for (
counter,
initialValue,
finalValue,
step
)


This command repeats the block of code enclosed between itslef and the *endfor command. Cycling is performed while verifing the initialValue and finalValue parameters on the counter value and adding step to it at every cycle excluding the first.



Parameters

counter
Name of the variable to use as the counter index of the cycle.

initialValue
The initial value to assign to the counter.

finalValue
The final value to assign to the counter.

step
The value to add to the counter at every cycle: it must be a positive, zero or negative integer value.



Return Values

This command doesn't return any data.

Remarks

To retrieve the value of the counter variable created with this command, you must use the *foridx command.

Example

*for(i, 1, 3, 1)

  *for(j, 3, 1, -1)
    *foridx(i)
    *foridx(j)
  *endfor(j)

*endfor(i)



The output of the example is 1 3 1 2 1 1 2 3 2 2 2 1 3 3 3 2 3 1

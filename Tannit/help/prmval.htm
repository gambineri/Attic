* prmval (
prmVarName
)


Retrieves the value of a prm variable, i.e. a variable declared and assigned in the current prm file.


Parameters

prmVarName
Name (case sensitive) of the variable in the prm file which has to be retrieved.



Return Values

A string representing the value of the requested variable in the current prm file.


Example

*prmval(tpldir)



 The sample code shown above returns the current template directory.

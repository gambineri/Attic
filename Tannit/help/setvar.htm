* setvar (
varName,
varValue
)


This command creates a variable named varName and sets its string formed value to varValue. The scope of such a variable is global; namely you declare a variable in a template and, after that, you can retrieve (or reset) its value in any subsequent point of the same template file or in any other one included with a *prex or similar command.


Parameters

varName
Name of the variable to create.

varValue
Value to assign to the variable being created.



Return Values

This command doesn't return any data.

Remarks

To retrieve the value of a variable created with this command, you must use the *getvar one.

Example

*getvar(myVar, *setvar(myVar, This is the value of myVar.))



The output of the example is the string This is the value of myVar.

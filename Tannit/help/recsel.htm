* recsel (
tqrName,
tqrField,
cnfSrc,
cnfField
)


Similarly to the *recval command, this one returns a string which is the tqrField field value in the current record of the tqrName specified tqr; furthermore, it adds the string " SELECTED" to the return value, if and only if data is equal to the one determined by the second couple of parameters: cnfSrc, cnfField which indicate respectively another tqr name and field name.


Parameters

tqrName
Name of the tqr from which the return value has to be extracted.

tqrField
Name of the field to be retrieved.

cnfSrc
Name of the tqr used to make the comparison; if this parameter is the string get, the comparison value is searched among the http GET parameters.

cnfField
Name of the field to compare.

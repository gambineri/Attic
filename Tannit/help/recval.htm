* recval (
tqrname,
fieldname
)


Retrieves from the specified tqr the value of the requested field in the current row.


Parameters

tqrname
Name of the Tannit Query Resultset (tqr) from which the data is to be retrieved.

fieldname
Name of the column whose data value is to be retrieved.



Return Values

A string representing the requested value.

Remarks



Example

*exeq(query, "SELECT * FROM table1")
*cycle(query)
 *recval(query, ID)
*endcycle(query)



 The sample code above returns the value of the field named ID for all the records contained in the table table1.

*cgipath()


Returns the complete pathname of the Tannit application: i.e. 'webroot' prm variable chained with 'cgidir' and 'cginame' ones.


Example

Suppose your prm file contains the following:

webroot=rootdir
cgidir=cgi-bin
cginame=tannit

by invoking the command:

*cgipath()

 then you obtain:

rootdir/cgi-bin/tannit

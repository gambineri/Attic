* tqrmove (
tqrName,
action,
[row]
)


Every tqr has an internal cursor over the rowset; this one has 1-based logic and it's used to make a specified record the current one.


Parameters

tqrName
Name of the tqr on which the command will operate.

action
Type of movement desired:
FIRST Moves the cursor to the first record.
LAST Moves the cursor to the last record.
NEXT Moves the cursor to the next record.
TO Moves the cursor to the record specified by the third parameter, row.

row
This parameter makes sense only if TO action is requested. See the description of the action parameter.



Return Values

This command doesn't return any data.

Example 

*exeq(q1, "SELECT * FROM TABLE1", 1, 1000)

*tqrmove(q1, LAST)
*tqrstat(q1, ACTUALROW)

*tqrmove(q1, FIRST)
*tqrstat(q1, ACTUALROW)

*tqrmove(q1, NEXT)
*tqrstat(q1, ACTUALROW)

*tqrmove(q1, TO, 3)
*tqrstat(q1, ACTUALROW)



The sample above returns the following results: 
1000
1
2
3

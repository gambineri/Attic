* fmtcur (
srcNum,
[decNum]
)


This command translates the integer srcNum value into Italian currency format with decNum decimal digits.


Parameters

srcNum
The input value

decNum
The desired number of decimals; it can be every nonnegative integer value.



Example 

*fmtcur(123456789, 0)
*fmtcur(123456789, 2)
*fmtcur(1000000, 10)



The output of the sample above is :
123.456.789
1.234.567,89
0,0001

* prex (
template
)


Includes the template specified in the parameter temporarily redirecting the processing to this file.


Parameters

template
Name of the template (without exetension) to include. This file must be placed in the current template directory.



Return Values

This command doesn't return any data.

Remarks

A variable created with the *setvar command before the instance of a prex commmand is also visible from within the included template

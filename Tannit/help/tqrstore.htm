*tqrstore(
tqrname,
nfields,
sep,
buffer
)


Creates a new tqr from an input string buffer.


Parameters

tqrname
Name of the Tannit Query Resultset (tqr) to be created.

nfields
Number of fields (columns) of the tqr record.

sep
Two characters string. First character is the field separator character use in the input string buffer. Second character is the record separator.

buffer
The input string buffer to be used as source of the tqr recordset.



Return Values


Remarks


Example

*tqrstore(foo, 4, @§, 1@bbb@ccc@ddd§2@eee@fff@ggg§)

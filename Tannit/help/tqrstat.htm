*tqrstat(
tqrName,
statFlag
)


Returns various kinds of statistics and info about the specified Tannit Query Resultset (tqr).


Parameters

tqrName
The name of the tqr which is requested info about.

statFlag
A string flag indicating the kind of info requested:

TOTRECS Total number of records actually contained in the tqr.
ACTUALROW The current value of a 1-based integer cursor on the records of the tqr. This value is -1 when the the cursor is positioned after the last row (EOF position); this value is 0 when the cursor is positioned before the first row (BOF position).
ROWSTOSTORE The requested number of rows when making the query.
STARTINGROW The requested 1-based index of the row from which to start when making the query and populating the tqr with records from db.
MORERECS When this flag is specified, the return value is a string containing a literal true or false, indicating respectively the existence (on the database) of other records more than the requested number.
SOURCECOUNT The return value of the command will be significative only if the tqr has been created from a database source. In this case, the return value indicates the number of records in the database source.
ACTUALPAGE Returns the current page number of the record subset loaded in the tqr.

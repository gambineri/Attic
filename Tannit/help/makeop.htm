*makeop (
expr,
[format]
)


Executes algebric operations not taking care of any kind of parenthesis.
It's possible to use the following operators: +, -, *, /, %


Parameters

expr
algebric expression to evaluate.

format
C-like format string: optional parameter.



Return Values

A string representing the result of the evaluation in decimal format if no format parameter is given.

Remarks

If you need to nest multiple expressions, you can do that by recursing the command itself.



Example

*makeop(*makeop(10 * 4) + *makeop(400 / *makeop(1 + 2 * 3 / 4) + 5) - 9, %.0f)

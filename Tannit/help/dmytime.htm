* dmytime (
utime,
[tag]
)


The command transforms a string representing a universal time (seconds past midnight of 1.1.1970) in a string of the type yyyy-mm-dd hh:mm:ss.
The optional tag parameter can assume d, m, y values indicating that the user request is limited respectively to the day, the month or the year only.


Parameters

utime
universal time to convert.

tag
optional parameter which can assume only d, m, y values, restricting the output only to the day, the month or the year.



Return Values

A formatted date-time of the input utime parameter as described above.

* getmad (
varName,
[defVal]
)


Returns a string representing the value of the get parameter whose name is varName actual value.
This command acts like the *get one, but doubles single and double quotes, if any.


Parameters

varName
Name of the get parameter to retrieve.

defVal
If the parameter is not found in the get string, the value contained in this parameter is returned, unless it'a + sign: in this case, (and if the parameter is found) the value returned is a string in which every non-alphanumeric ascii character is substitued with the relative two-digits hex sequence %xx. If the parameter has not been found, the value returned is the empty string.

* traceu (
feature
)


Retrieves various informations about the current user agent.


Parameters

feature
String which represents the nature of the information that is being retrieved:


brname browser name.
brver browser version.
osname operative system name.
ipnum ip address.
ipname host name (if the web server exports this info).



Return Values

A string representing the content of the requested information.


Example

Browser = *traceu(brname)
Browser version = *traceu(brver)
OS Name = *traceu(osname)
IP Address = *traceu(ipnum)
Hostname = *traceu(ipname)



 This produces the output:


Browser = MS Internet Explorer
Browser version = 4.01
OS Name = MS Windows NT
IP Address = 194.0.0.11
Hostname =

* getvar (
varName,
varDefault (opt)
)


This command retrieves the value created and set with a *setvar command.


Parameters

varName
Name of the variable whose value has to be retrieved.

varDefault
Default value used in case the variable is not defined or in case the variable is empty.



Return Values

This command returns a string representing the value of the variable that is being retrieved.

Remarks

To retrieve the value of a variable with this command, you must create it with the *setvar one.

Example

*getvar(myVar, *setvar(myVar, This is the value of myVar.))



The output of the example is the string This is the value of myVar.

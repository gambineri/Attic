PRM files



A prm file is a text file which represents the initialization file for the current running instance of Tannit.

The schema of this file is:


variablename=variablevalue
variablename=variablevalue
................

# This is a comment line ................

Every variable name declaration is case sensitive (namely, myvar differs from myVar).
To retrieve in a certain template the value of a prm variable, you must use the *prmval command.
